COMMISSARIAT DE RIVEMONT

Démarches administratives :

  • Signaler un vol ou une perte de papiers
  • Obtenir un récépissé de dépôt de plainte
  • Télécharger une attestation de dépôt
  • Suivre le traitement d’une réclamation
  • Modifier ou compléter une plainte existante

Données personnelles & sécurité :

  • Demander l’accès à ses données personnelles (RGPD)
  • Signaler une usurpation d’identité
  • Faire une demande de casier judiciaire

Demandes judiciaires :

  • Faire une demande de médiation pénale
  • Suivre une convocation au tribunal
  • Prendre contact avec un officier de liaison

Infractions / circulation :

  • Suivre le traitement d’un retrait de permis
  • Payer une amende forfaitaire
  • Demander un échelonnement d’amende
  • Demander une copie de procès-verbal
  • Contester une contravention
  • Demander un cliché radar

Vie du commissariat :

  • Poser une question à un agent de permanence
  • Prendre un rendez-vous avec un agent
  • Déclarer un trouble à l’ordre public
  • Faire remonter un comportement suspect dans le quartier
